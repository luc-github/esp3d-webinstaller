🚀 New Features

Bluetooth PIN Code Support

Users can now customize the Bluetooth PIN code (4-digit or 6-digit) for both legacy and BLE connections. This allows easier pairing with CNC controllers or other devices.

Dedicated Connection Status Screen

A new dedicated screen has been added to display the current Bluetooth connection status, including the ability to disconnect and forget the paired device.

Factory Partition

Implement factory partition with custom bootloader to allow for easier recovery and re-flashing of the firmware in case of issues during updates or development. It allows to choose booting partition if several are available and update firmware from SD card without needing to connect to a computer.

The Factory partition is accessible when pressing Right button (Button 3) before and during boot, a beep at press, inform you get the bootloader get the request, then you can release and hear a new melody, then factory partition is displayed.

If no button is pressed during boot, the firmware will boot from the normal partition as usual.

User documentation will be updated later to explain how to use the factory partition and how to do firmware updates from SD card.

[!WARNING]

This new feature breaks the previous partition scheme and requires a full re-flash (bootloader/partitions/firmware) of the firmware to update to this version.

On 4MB device there is not more space for OTA updates, so to do update you need to use the factory partition and flash the firmware from SD card. The firmware update file will be available as download from web installer page, and you can copy it to the SD card, then use the factory partition to flash it.

🐛 Bug Fixes

  • Unwanted Warnings Removed: Several unwanted error/warning messages from the ESP-IDF libraries have been disabled to improve the overall logging and avoid noise on console output.

💪 Improvements

  • Bluetooth Connection Stability Improvements: Several changes have been made to improve the stability and responsiveness of Bluetooth connections, including the use of new priority APIs for real-time commands.

📝 Other Changes

  • Bluetooth PIN Code Defaults Updated: The default Bluetooth PIN code has been changed to '0000' for legacy connections and '000000' for BLE connections to align with common industry standards.
